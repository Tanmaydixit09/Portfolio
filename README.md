# Portfolio

This repository contains my personal portfolio built with React and Vite.

Background and structure
- The project is a React + Vite application using `@vitejs/plugin-react` and TailwindCSS for styling.
- Source files live under `src/` and static assets under `public/`.

Developer notes
- Run `npm install` to install dependencies.
- Development server: `npm run dev`.
- Build: `npm run build` and preview via `npm run preview`.

If you're expanding or adapting this project as a template, consider the TypeScript template from Vite for stricter linting and type-aware rules.
